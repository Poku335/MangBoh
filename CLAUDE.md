# ReadManboh Project

## ⚡ อ่านก่อนทำงานทุกครั้ง

อ่าน `C:/nova/hotcache.md` ก่อนเสมอ — มีสรุป context ล่าสุดของโปรเจ็คไว้แล้ว ถ้าคำตอบอยู่ในนั้นให้ใช้ได้เลยโดยไม่ต้องอ่านไฟล์เพิ่ม

ถ้าต้องการข้อมูลเพิ่มเติม: อ่าน `C:/nova/index.md` เพื่อหาหน้าที่เกี่ยวข้อง แล้วค่อยเปิด `C:/nova/pages/` ที่ต้องการ

## Stack

Next.js 15 · React 19 · Prisma 5 · PostgreSQL · NextAuth 5 (beta) · Tailwind 3 · TypeScript

## โครงสร้าง

- `app/` — Next.js App Router (pages, API routes, layouts)
- `components/` — React components
- `lib/` — auth, prisma client, session helpers
- `prisma/` — schema และ migrations
- `C:/nova/` — wiki ความรู้ใช้ร่วมกันหลาย project (อ่านด้วย `C:/nova/hotcache.md`)
